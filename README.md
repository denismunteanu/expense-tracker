# Expense Tracker

This is a CLI-Based Expense Tracker. You can use it to store your expenses into categories and save all your data in one file, so when you come back you can start from where you left off.

## How to use

Simply enter any number from 1 to 10 based on which of the following options in the "HOME" section you want to execute:

1. Add an Expense
2. Add a category to store expenses
3. Move any expense into a category
4. Show all expenses neatly
5. Calculate the total cost of all expenses
6. Remove any expense
7. Remove any category
8. Change your username
9. Change your preferred currency
10. Save all your Data & Exit

## How to run

Simply run the script from any IDE or terminal:

```
python expense_tracker.py
```

The program will start with an introduction. Your data is saved in a `data.json` file in the same folder as the script.
